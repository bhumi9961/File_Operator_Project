# Personal Journal Manager 📝

A simple command-line based journal application built using Python.
This program allows users to write, view, search, and delete journal entries stored in a text file.

---

## 🚀 Features

* Add a new journal entry with timestamp
* View all saved entries
* Search for specific words in entries
* Delete all entries (removes the file)
* Handles errors like:

  * File not found
  * Permission issues
  * Invalid input

---

## 🛠️ Technologies Used

* Python
* File Handling (`.txt`)
* Modules:

  * `datetime`
  * `os`

---

## 📂 File Structure

```
project/
│
├── journal.txt        # Stores all journal entries
├── main.py            # Main program file
└── README.md          # Project documentation
```

---

## ▶️ How to Run

1. Make sure Python is installed
2. Run the script:

```
python main.py
```

---

## 📌 Menu Options

```
1. Add a New Entry
2. View All Entries
3. Search For an Entry
4. Delete All Entries
5. Exit
```

---

## ⚠️ Notes

* Entries are stored in `journal.txt`
* Deleting entries removes the file permanently
* Searching is case-insensitive
